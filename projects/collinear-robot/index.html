Collinear Robot

As a capstone project (Gymnasiearbete), I was one in a group of three who built a balancing robot. I was responsible for writing the code. This entailed making an estimate of the current angle, interacting with a PID controller, communicating with the motor controllers, as well as allowing manual control through a Playstation 4 controller.

The code is available on Github.
